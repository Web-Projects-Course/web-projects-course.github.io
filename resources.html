Software

We will use the following tools to create our web projects.

Brackets
Open-source, cross-platform text editor created by Adobe. It designed specifically for working with Web documents.
Cyberduck
File Transfer Protocol(FTP) client. We will use FTP to move files from our local machines to servers.
Photoshop
Any image editor capable of basic editing, adjusting and exporting can be used. For demostrations we will use Photoshop.

Services

We will rely on a couple services for online collaboration and communication.

GitHub
GitHub is a service for hosting Git repositories. We will use it to share some code.
Slack
Slack will be used for onlconversations.
CodePen
A great way to test out code.
