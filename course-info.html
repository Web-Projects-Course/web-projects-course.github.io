Labeled a village, a frontier, a highway, a commons, a global brain, the cloud, and a series of tubes, the internet and its constituent technologies have been described by many metaphors. Used as explantory tools, metaphors are often useful in grounding abstract technical concepts, guiding users toward possible uses. They can also serve as political or artistics contraints.

This course explores the theoretical and practical framework of the internet and its most common interface, the web. We will use metaphor as our compass in navigating this vast subject. The heart of the course will be a series web-based projects using HTML, CSS, and JavaScript. Students are encourage to interpret the web as an experimental platform and as a subject of contemporary culture.

Objectives

By the end of this course student will...

...create sophisticated web-based work using HTML, CSS and Javascript

...be able to speak clearly about web-based work, placing it in a greater context of internet culture and design

...develop skills for learning new technologies through community and individual effort


Overview

In the first half of the course, we focus on learning HTML, CSS and JavaScript. Projects and exercises will focus on this task. In the second half, we look at additional web tools that relate specifically to the work and interests of the students, culminating in a final project of the student's choosing.

Learning together

Learning new technologies can be daunting and so a key element of this course will be learning to use each other and the broader internet community as a resource. In class we will conduct short demonstrations followed by hands-on collaborative exercises. Outside of class we will share found resources and techniques in an online forum.

Artistic possibilities

Throughout the course we will address how our work fits into the broader conversation on the internet. We will survey artists and designers using the web as an expressive medium and we will also look at ways to push web-based work beyond the screen.

Expectations

Attendance

Because collaborative learning is so central to this course, attendance is critical to your success. If sickness or other personal reasons prevent your attendance please contact the instructor as soon as possible.

Plagerism

Issues of appropriation, reproduction and reuse will be a conceptual problem of this course. In the interest of academic honesty, students should keep track of the sources of the materials, text and code they use in their coursework. Please include a list of sources and citations used with any completed work.

Please see RISD's Academic Code of Conduct for a detail description of school policies.

Required materials and tools

In order to complete the work for this course students will need to have a laptop. We will be using mostly free and open source software but some use of image editing software, like Adobe Photoshop will be required.

There is no required text for this class.

A full list of software and other resources can be found at the resources page.
